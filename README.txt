PDF compressor
This terminal can compress .pdf files

How to use?
1. Open pdf_compressor.py
2. Right click and choose "Open in Integrated Terminal"
3. Install PDFNetPython3 library, "pip install PDFNetPython3" in terminal
4. Parse "python pdf_compressor.py input_file.pdf output_file.pdf" in terminal
The last argument (output_file.pdf) is optional. If there is none, the code will automatically output the compressed file as input_file_compressed.pdf 
5. Enjoy!
